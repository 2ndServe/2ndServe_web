# Second Serve Website

Second Serve is a student-led nonprofit dedicated to giving used pickleballs a second life instead of sending them to landfills. This site is a modern, responsive, easy-to-maintain web presence built with HTML, CSS, and vanilla JavaScript.

## Features

- Multi-page website for home, about, donate, impact, and contact
- Mobile-first responsive design
- Accessible semantic HTML and keyboard-friendly navigation
- Smooth scroll, fade-in on scroll, hover effects, and animated counters
- Contact form with client-side validation and success message
- GitHub Pages deployment ready

## Run locally

1. Clone the repository.
2. Open the project folder in your code editor.
3. Open any HTML file in the browser, or use a local server extension.

## Deploy with GitHub Pages

1. Push the repository to GitHub.
2. Ensure `.github/workflows/pages.yml` exists.
3. In GitHub repository settings, enable Pages and select the `gh-pages` branch or the repository's Pages settings may automatically deploy from `gh-pages`.
